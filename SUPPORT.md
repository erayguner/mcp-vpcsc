## Asking questions

- **GitHub Issues** — [open an issue](https://github.com/erayguner/mcp-vpcsc/issues/new/choose) for bug reports and feature requests using the provided templates.
- **GitHub Discussions** — use [Discussions](https://github.com/erayguner/mcp-vpcsc/discussions) for questions, ideas, and general conversation.

## Reporting bugs

Use the [bug report template](https://github.com/erayguner/mcp-vpcsc/issues/new?template=bug_report.yml). Include:

- Version of vpcsc-mcp
- Transport (stdio, streamable-http, SSE)
- Python version
- Steps to reproduce
- Expected vs actual behaviour
- Relevant logs (redact sensitive information)

## Security vulnerabilities

Do **not** open a public issue. Follow the [security policy](SECURITY.md) for responsible disclosure.
